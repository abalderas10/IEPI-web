## Instalación Local

```bash
# Clonar el repositorio
git clone https://github.com/abalderas10/IEPI-web.git

# Navegar al directorio
cd IEPI-web

# Abrir en el navegador (servir con un servidor local)
# Opción 1: Python 3
python -m http.server 8000

# Opción 2: Node.js (si tienes http-server)
npx http-server
```

Luego abre `http://localhost:8000` en tu navegador.

## Despliegue

Este sitio se despliega automáticamente con GitHub Pages cuando se realizan cambios en la rama `main`.

### Configurar tu propio dominio

1. Ve a la configuración del repositorio en GitHub
2. Busca "Pages" en el menú lateral
3. Ingresa tu dominio en el campo "Custom domain"
4. Configura los registros DNS en tu proveedor de dominio
